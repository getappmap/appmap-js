See your code run.

AppMap records what actually happens at runtime — every request, query, and call — and renders it as a live trace. This visual is the real thing, animating a request as it flows through the stack.

Scenarios

Each <appmap-live-trace> is self-contained — set the scenario attribute to swap the story.

scenario="web"
scenario="background"

Embed with <script src="appmap-live-trace.js"></script> then <appmap-live-trace></appmap-live-trace>. No build step, no dependencies. Honors prefers-reduced-motion.
